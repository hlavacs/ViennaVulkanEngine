◆ veEventLifeTime

enum ve::veEventLifeTime

Lifetime of an event.

A once-event will be destroyed right after event processing. A continuous event will remain in the event list and will be sent to the listeners in the next loop.

Enumerator
VE_EVENT_LIFETIME_ONCE 

One time event.

VE_EVENT_LIFETIME_CONTINUOUS 

Continuous event.

◆ veEventSubsystem

enum ve::veEventSubsystem

Enumerates possible subsystems.

A subsystem determines the scope of events and how event data are interpreted. For example, GLFW is a subsystem using its own event types and keyboard IDs. Event listeners can use the subsystem to determine whether they can even understand the event.

Enumerator
VE_EVENT_SUBSYSTEM_GENERIC 

No subsystem given.

VE_EVENT_SUBSYSTEM_GLFW 

GLFW.

Enumerates event types.
