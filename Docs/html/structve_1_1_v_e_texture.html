VETexture constructor from a list of textures.

Create a VETexture from a list of textures. The textures must lie in the same directory and are stored in a texture array. This can be used also as a 2D texture or a cube map.

Parameters
[in]nameThe name of the mesh.
[in]basedirName of the directory the files are in.
[in]texNamesList of filenames of the textures.
[in]flagsVulkan flags for creating the textures.
[in]viewTypeVulkan view tape for the image view.
