Detailed Description

A classical forward renderer.

This renderer first clears the framebuffer, then starts rendering each entity one by one.

Member Function Documentation

◆ drawFrame()

void ve::VERendererForward::drawFrame ( )
protectedvirtual

Draw the frame.

  • wait for draw completion using a fence, so there is at least one frame in the swapchain
  • acquire the next image from the swap chain
  • if there is no command buffer yet, record one with the current scene
  • submit it to the queue
