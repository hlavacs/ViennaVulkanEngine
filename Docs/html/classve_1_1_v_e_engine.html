Detailed Description

The engine core class.

VEEngine is the core of the whole engine. It is responsible for creating instances of the other management classes VEWindow, VERenderer, and VESceneManager, and running the render loop. In the render loop, it also handles events and sends them to the registered event listeners.

Call all event listeners.

This function calls all event listeners and gives them a specific event. The listeners can react to the event or not. If one listener retursn true, then event is "consumed" by this listener, and the function stops. Otherwise it continues and goes on calling listeners.

Parameters
[in]dtDelta time that passed by since the last loop.
[in]eventThe event that should be passed to all listeners.
[in]listThe list of registered event listeners, that should receive this event

void ve::VEEngine::initEngine ( )

Initialize the engine.

Creates a VEREnderer, VESceneManager, VEWindow. Then initializes Vulkan and gets the Vulkan instance. After this, initializes VERenderer, VESceneManager and VEWindow. Then it registers default event listeners.

◆ run()

void ve::VEEngine::run ( )
virtual

The main render loop.

This is the main render loop. It performs time measurements, runs the event processing, checks whether the window size has changed (if so, informs the VEREnderer), and asks the VERenderer to draw and present one frame.
