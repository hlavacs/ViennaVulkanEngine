Detailed Description

Base class of all event listeners.

An event listener is a class that is sent events from the engine. The listener can examine the event and can decide to act on it. Certain events can be consumed by returning true, like a keyboard stroke. Others like onFramStarted cannot be consumed and are always sent to all event listeners.
